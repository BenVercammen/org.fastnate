Fastnate

The Offline SQL Generator

Download this project as a .zip file Download this project as a tar.gz file

Fastnate is a JPA tool to generate SQL statements without having access to a database. Its main use case is to create test data for databases during development time.

We are currently working on version 1.0, so stay tuned...
